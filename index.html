Open Video Analysis Framework

Video content is challenging to research because of the sheer volume of data. In order to ease this burden a little, I've started an open source framework (OpenVAF) with the intention of it becoming a project similar to what OpenCV is for Computer Vision.

Unreal AI Labs

Unreal Engine is a massively powerful rendering engine, with woefully inadequate support for AI. In this project the intention is to add to the AI functionality of UE4 in the area of machine learning, computer vision and in the basics like navigation and markup.
